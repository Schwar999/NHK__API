# NHK__API
Google Apps Scriptを使ったNHKの番組表が届くシステム
GASの特徴であるトリガーシステムのお陰で番組表との相性は良い。
メールで入力情報を送ると番組出力させることも可能で、今後取り組んでいきたい。

main.gs

APIに欲しい情報を配列から読み込み、ループさせて出力させた。
日時、ジャンル等を配列にPushで組み込んだ後は必要な情報のみ取り出しgmail.gsに受け渡す。

  var str_service = str_list+'.'+service; //strで文字化（'.'がコードで置けないため一度strで置いてevalで変換）

Google Apps Script(以下、GAS)の特徴として、配列に組み込む際、"."がエラーとして読み込んでしまうためstringで一度変換し、文をURLとして読み込まなければならない
上記の方法で行ったが正しいやり方があるなら教えてほしい。

辞書を使えばもっとクールにできたかもしれないが現在、この形で動いている。
